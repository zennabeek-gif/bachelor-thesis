# bachelor-thesis
This repository contains the relevant code, data, and the analysis file used in this bachelor thesis. Exploring User-Initiated vs. System-Initiated Assistance During Cognitive Rehabilitation Tasks

analysis.py: contains the Python scripts used to analyse the questionnaire data and generate a visualisation.

results.pdf: contains the questionnaire responses and interview summaries per participant.

prototype_code: contains the relevant code used for the PsychoPy experiment.
- difficulty.py: assigns a difficulty level based on the calibration round.
- help_generation.py: generates hints for both help conditions.
- system_intitiated_triggers.py: determines when automatic assistance is offered.
